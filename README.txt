1.各資料夾的檔案性質
(1)自行開發之程式碼:系統執行之程式碼
(2)所有文件:專案及系統介紹
(3)執行專案所須之完整開發軟體工具:此系統開發與執行所使用之工具
--------------------
2.開發軟體工具安裝之順序步驟與設定
(1)安裝Visual Studio Code
(2)安裝Appserv
(3)安裝Python3
--------------------
3.自行開發程式碼之安裝設定
(1)開啟http://localhost/phpMyAdmin並匯入project.sql完成資料庫之建立
(2)開啟資料夾之AppServ/www/並將自行開發之程式碼中所有檔案貼上
--------------------
4.測試安裝成功之步驟
(1)開啟http://localhost/php/home.php如可正常顯示頁面及安裝成功
(2)如要登入，選取右上角之選單之登入輸入帳號(root)及密碼(root)即可
--------------------
